# Bigdata
# 🚇 2025년 9월 서울 지하철 승하차 인원 데이터 분석  
---

## 📌 프로젝트 개요
이 프로젝트는 **2025년 9월 서울 지하철 각 역별 승차/하차 총인원 데이터**를 바탕으로  
이용량 패턴을 분석한 데이터 분석 프로젝트입니다.

시간대별 데이터는 제공되지 않아 **일자별·노선별·역별 총 승하차량 중심으로 분석**을 진행했습니다.

## 🗂 데이터 정보
- **데이터 출처:** 서울 열린데이터 광장  
- **기간:** 2025년 9월 1일 ~ 2025년 9월 30일  
- **데이터 형식:** CSV  
- **구성 컬럼:**  
  - `날짜`
  - `호선명`
  - `역명`
  - `승차총승객수`
  - `하차총승객수`

⚠️ **시간대 데이터(출근/퇴근 시간별 이용량)는 포함되지 않음**

---

## 🛠 기술 스택
- **Language:** Python 3.10+
- **Libraries:**  
  - pandas – 데이터 처리  
  - numpy – 계산  
  - seaborn / matplotlib – 시각화  

---

## 📊 주요 분석 결과

### 1️⃣ 승차/하차 이용량 TOP 10 역
- 서울 주요 환승역과 업무지구가 상위권에 집중  
- 2호선·9호선의 비중이 큼

### 2️⃣ 노선별 이용량 분포
- 2호선이 가장 많은 승하차량을 기록  
- 5호선·7호선 등에서도 꾸준한 이용량 확인

### 3️⃣ 날짜별 이용 추이
- 평일이 주말보다 높은 경향  
- 특정 주말/공휴일에 이용량 급감 패턴 존재
